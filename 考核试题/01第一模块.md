    - 问修改前后info的id是否改变
    - 问修改前后info['alex']['hobbies']的id是否改变
  ```
  info={
  	'egon':{'pwd':'123','hobbies':['play','music','read']},
  	'alex':{'pwd':'1234','hobbies':['music','read']},
  	'wupeiqi':{'pwd':'135','hobbies':['read',]},
  }

  info['alex']['hobbies'].append('play')
  ```

  正确答案：都不变
  ```
    >>> id(info)
    4320179096
    >>> id(info['alex']['hobbies'])
    4320029872
    >>>
    >>> info['alex']['hobbies'].append('play')
    >>>
    >>> id(info)
    4320179096
    >>> id(info['alex']['hobbies'])
    4320029872
  ```
